# Leaf Disease Detection using KNN

Machine Learning project that detects apple leaf diseases.

Diseases Supported
- Black Rot
- Cedar Rust
- Apple Scab

Technologies
- Python
- Flask
- OpenCV
- Scikit-Learn
- Bootstrap UI

Features
- Upload leaf image
- Detect disease
- Show disease description
- Modern UI
- KNN Machine Learning model

Run Project

pip install -r requirements.txt

python train_model.py


python app.py
